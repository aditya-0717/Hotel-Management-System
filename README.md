# Hotel Management System (Java)

A Java-based Hotel Management System developed using Core Java and Swing.
This project helps manage hotel operations such as room booking, customer
details, employee management, and billing.

## Features
- Login System
- Room Management
- Customer Check-in & Check-out
- Employee Management
- Search Room
- Billing System

## Technologies Used
- Java
- Swing
- AWT
- VS Code
